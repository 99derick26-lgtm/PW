Rules to remember:

- One joined guild plus one created/led guild is allowed.
- Creating a guild must not require leaving a joined guild.
- Joining another guild requires leaving the current joined guild.
- Guild state, guild vault, jail, loot, chat, and member management are server-owned.
- Guild LAND collection sends rewards to the server guild vault.
- Fight All must fight the 8 visible arena opponents.
- Pets use percentage scaling from final avatar combat stats, not independent level curves.
- Ignore AFO summary text when it conflicts with current Pixel War rules.
